json_res

result string in json format.
app_path

data_path

/data/app/<codePath>
sd0_path

sd1_path

void
HCFS_pin_path ( char * json_res,
char * pin_path )

Pin a file so that it will never be replaced when doing cache replacement. If the given (pin_path) is a directory, HCFS_pin_path() will recursively pin all files and files in subdirectories.

json_res

result string in json format.
pin_path

a valid pathname (cloud be a file or a directory).
